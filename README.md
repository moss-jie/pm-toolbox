# 🧰 默默的产品经理工具箱

> 一个产品经理学习编程的实战项目——用代码解决 PM 日常工作痛点。

---

## 🛠️ 工具列表

| 工具 | 说明 | 打开方式 |
|------|------|---------|
| 📝 **每日日志生成器** | 自动生成 Markdown 格式的工作日志 | `node src/hello.js` |
| 🎨 **个人名片页** | HTML + CSS 毛玻璃风格名片 | 双击 `src/card/index.html` |
| 📊 **需求优先级打分器** | 加权评分 + 可视化图表 + 主题切换 | 双击 `src/calculator/index.html` |

---

## 🚀 怎么运行

```bash
# 安装依赖（只需要一次）
npm install

# 运行日志生成器
npm run hello
```

HTML 页面直接双击文件即可在浏览器打开。
